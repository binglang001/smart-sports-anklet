# 人体姿态检测与步数检测算法分析报告

## 摘要

本报告基于6篇学术论文，系统分析了人体姿态检测、步数检测和跌倒检测的完整算法流程。报告重点提取了基于腰部加速度传感器（采样率50Hz）的步数检测算法、坐姿/站立检测算法和跌倒检测算法，并汇总了论文中的关键公式、参数阈值及示意图说明。所有算法内容均直接来源于论文原文，确保信息的真实性和完整性。

---

## 一、步数检测完整算法流程（采样率50Hz，传感器佩戴在后背腰椎处）

### 第1步：原始数据采集与预处理

**论文来源**：

- 论文标题：《基于腰部MEMS加速度计的多阈值步数检测算法》
- 作者：蒋博（通讯作者）、付乐乐
- 单位：重庆邮电大学光电工程学院
- 发表期刊：《传感器与微系统》（注：论文内容显示为重庆邮电大学光电工程学院，具体期刊名称待确认）

**具体算法描述**：

本实验选用SCR系列加速度计，实验人员把加速度计佩戴在腰部，在运动场进行100米直线运动步数检测实验。采集数据频率为50Hz，利用去除重力加速度g的xyz三轴合加速度的模值作为初始检测数据。成年人正常行走中，步行速度大约0.5-2m/s，步距大约0.6-0.7m，每迈出一步的频率大约1-2步每秒；在快速奔跑时，跑步速度可达5-10m/s，步距可达1-2m，每迈出一步的频率大约3-4步每秒。

行人行走时左右双脚交换迈出步子的运动特征，身体腰部的运动信息呈现出一种近似正弦曲线的周期变化趋势。因此，把加速度计安置在人员躯干的腰椎部，对人员躯干周期性运动变化情况进行分析，采集行人走动时的加速度值，设置多阈值判断条件，实现对行人运动步数的精准检测。

**输入参数**：

- 采样频率：50Hz（50次/秒）
- 传感器类型：SCR系列加速度计
- 传感器佩戴位置：腰部腰椎处
- 合加速度计算公式：A = √(Ax² + Ay² + Az²)

**输出参数**：

- 合加速度模值（单位：m/s²或g）
- 步数检测结果（整数）

### 第2步：重力加速度去除

**论文来源**：

- 论文标题：《分段双向去除反向重力加速度算法》
- 作者：李兴、侯振杰（通讯作者）、梁久祯、常兴治
- 单位：常州大学信息科学与工程学院、江苏省物联网移动互联技术工程重点实验室、常州信息职业技术学院
- 发表期刊：《计算机辅助设计与图形学学报》，2019年，第31卷，第4期
- DOI：10.3724/SP.J.1089.2019.17344

**具体算法描述**：

由于近地面地球引力的影响，加速度传感器采集到的传感器加速度是反向重力加速度与线性加速度的矢量和。只有线性加速度与人体的运动相关，反向重力加速度的存在导致传感器加速度与人体运动的关联性不高，进而影响人体行为识别的精度。

分段双向去除反向重力加速度的流程主要包括以下两个步骤：

**步骤2.1：基于静止点的数据分段法**

静止的物理含义是指物体相对于地面，速度与加速度都为零的一种状态，使用加速度传感器可以直接检测出加速度为零的状态。当人体部位处于静止状态时，线性加速度为零，此时采集到的数据点为一类特殊数据点，此类数据点中的三轴传感器加速度是反向重力加速度在当前时刻传感器坐标系下的投影，本文将这样的数据点称作静止点。将静止点作为数据端点，把六轴力学传感器采集到的运动数据进行分段。分段后每段运动数据的起始点与末尾点都为静止点，满足使用角度法或双向角度法对数据段进行去除反向重力加速度的前提。

**步骤2.2：双向角度法**

与角度法不同，双向角度法首先从前往后积分计算反向重力加速度得到线性加速度as，然后从后往前积分计算反向重力加速度得到线性加速度at。由于两种线性加速度数据准确性的变化趋势相反，取两者平均可得到更准确的线性加速度。

**关键公式**：

**公式1：加速度误差模型**
a'(i) = a(i) + g(i) + an(i) + ae(i)

其中：

- a'(i)：第i个数据点中的三轴传感器加速度向量（单位：m/s²）
- a(i)：真实三轴线性加速度向量（单位：m/s²）
- g(i)：重力加速度向量（单位：m/s²）
- an(i)：三轴加速度随机误差向量（单位：m/s²）
- ae(i)：三轴加速度仪器误差向量（单位：m/s²）

**公式2：角度计算值误差模型**
e'(i) = e(i) + en(i) + ee(i)

其中：

- e(i)：角度真实值（单位：度或弧度）
- en(i)：由传感器角速度的随机误差ωn(i)通过积分积累而来的误差值（单位：度或弧度）
- ee(i)：由仪器误差ωe(i)通过积分积累而来的误差值（单位：度或弧度）

**算法流程**：

```
输入：原始传感器加速度a'(i)，角速度ω'(i)
输出：线性加速度a(i)

1. 静止点检测：
   - 检测一段时间内加速度为零的数据点作为静止点
   - 静止点判断条件：|a'| < 阈值（通常小于0.1g）

2. 基于静止点数据分段：
   - 以静止点为端点将运动数据分段
   - 每段数据的起始点和末尾点都为静止点

3. 双向角度法去除重力加速度：
   3.1 从前往后积分：
       - 使用角速度ω'(i)从前往后积分计算角度e'(i)
       - 根据角度计算反向重力加速度g'
       - 计算线性加速度as = a' - g'

   3.2 从后往前积分：
       - 使用角速度ω'(i)从后往前积分计算角度e''(i)
       - 根据角度计算反向重力加速度g''
       - 计算线性加速度at = a' - g''

   3.3 取平均：
       - a(i) = (as + at) / 2

4. 拼接各段线性加速度得到完整结果
```

### 第3步：多阈值步数检测算法

**论文来源**：

- 论文标题：《基于腰部MEMS加速度计的多阈值步数检测算法》
- 作者：蒋博（通讯作者）、付乐乐
- 单位：重庆邮电大学光电工程学院
- 作者简介显示：蒋博（1991—），男，重庆梁平人，硕士研究生；付乐乐（1992—），男，江苏徐州人，硕士研究生

**具体算法描述**：

多阈值步数检测算法流程图如图1所示，分为以下5个步骤：

**步骤1（初始化）**：

首先初始化步数Step=0，标志位Flag=1，建立一个实时的数组D[n]，接着设定一个窗口大小为7的滑动缓存窗口C[k]，(k<n)，缓存窗口每一次滑动右移更新一个数C[k]=C[k+1]。

**输入参数**：

- 窗口大小：7（固定值）
- 数组D[n]：存储实时采集的合加速度数据
- 初始步数：Step = 0
- 初始标志位：Flag = 1

**步骤2（波峰检测）**：

然后检测Flag，若Flag=1，则检测波峰阈值条件：缓存窗口C[k]中的7个数都大于波峰阈值Tmax，且中间第4个数比其余6个数都大。如果没有达到波峰阈值条件，则返回缓存窗口，并更新一个数，再次检测波峰阈值条件；如果达到波峰阈值要求，就设置标志位Flag=2。

**波峰检测条件**：

```
条件1：C[0] > Tmax 且 C[1] > Tmax 且 C[2] > Tmax 且 C[3] > Tmax 且 C[4] > Tmax 且 C[5] > Tmax 且 C[6] > Tmax
条件2：C[3] > C[0] 且 C[3] > C[1] 且 C[3] > C[2] 且 C[3] > C[4] 且 C[3] > C[5] 且 C[3] > C[6]
同时满足条件1和条件2时：Flag = 2
```

**参数说明**：

- Tmax：波峰阈值（单位：m/s²或g），需要根据实际标定设定
- 缓存窗口C[k]长度为7，C[3]为中间元素

**步骤3（过零点检测）**：

接着检测Flag，若Flag=2，则检测过零点阈值条件：缓存窗口C[k]中的中间第4个数小于0，且第3个数大于0。如果不符合过零点要求，则返回缓存窗口；如果符合过零点要求，则设置标志位Flag=3。

**过零点检测条件**：

```
条件1：C[3] < 0
条件2：C[2] > 0
同时满足条件1和条件2时：Flag = 3
```

**参数说明**：

- 过零点阈值为0（合加速度相对于重力加速度的偏差）

**步骤4（波谷检测）**：

接下来检测Flag，若Flag=3，则检测波谷条件：缓存窗口C[k]中的7个数都小于波谷阈值Tmin，且中间第4个数小于其余6个数。如果不满足波谷条件，则返回缓存窗口；如果满足波谷，则重置标志位Flag=1，步数Step加一，完成一个步数检测。

**波谷检测条件**：

```
条件1：C[0] < Tmin 且 C[1] < Tmin 且 C[2] < Tmin 且 C[3] < Tmin 且 C[4] < Tmin 且 C[5] < Tmin 且 C[6] < Tmin
条件2：C[3] < C[0] 且 C[3] < C[1] 且 C[3] < C[2] 且 C[3] < C[4] 且 C[3] < C[5] 且 C[3] < C[6]
同时满足条件1和条件2时：
    Flag = 1
    Step = Step + 1
```

**参数说明**：

- Tmin：波谷阈值（单位：m/s²或g），需要根据实际标定设定，通常为负值

**步骤5（结束判断）**：

最后检测数组D[n]是否载入完毕，如果没有载入完毕，则返回缓存窗口；如果载入完毕，则输出步数Step，结束整个步数检测流程。

**算法流程图**：

```
开始
  |
  v
初始化：Step=0, Flag=1, D[n], C[7]
  |
  v
检测Flag==1? --是--> 波峰检测：C[0-6]>Tmax 且 C[3]最大
  |                         |
  |                         是---> Flag=2
  |                         |
  否                         v
  |                    检测Flag==2?
  |                      |
  否---> 过零点检测：C[3]<0 且 C[2]>0
            |                      |
            是---> Flag=3          |
            |                      v
            v                 检测Flag==3?
            |                      |
            否                  波谷检测：C[0-6]<Tmin 且 C[3]最小
            |                      |
            v                      是---> Flag=1, Step++
            |                      |
            否                     v
            v                      |
  D[n]载完? <-------------------返回缓存窗口
  |
  是
  |
  v
输出Step，结束
```

### 第4步：步数计算公式

**论文来源**：同第3步

**具体公式**：

本文采用依次检测放置于腰部的加速度计波峰、过零点与波谷三个阈值的方法，设定标志位Flag，在一个步态周期内，从波峰值开始检测，再检测过零点，接着检测波谷值，从而最终完成一个步数Step的检测。

**步数检测核心公式**：

```
Step = Σ(满足完整步态周期的次数)

其中完整步态周期定义为：
1. 波峰条件满足（Flag从1变为2）
2. 过零点条件满足（Flag从2变为3）
3. 波谷条件满足（Flag从3变为1），同时Step++
```

### 第5步：实验结果验证

**论文来源**：同第3步

**实验结果**：

在慢走、正常行走、快走、慢跑与快跑5种运动状态下，以实验人员实际运动步数为参照，对本文提出的步数检测算法与传统的波峰、过零点检测算法进行对比验证。多种运动模式下前进100米的步数检测结果如下表所示：

| 运动模式 | 实际步数 | 波峰检测 | 过零点检测 | 多阈值检测 |
|---------|---------|----------|-----------|-----------|
| 慢走    | 157     | 146(93%) | 147(94%)  | 156(99%)  |
| 正常    | 143     | 135(94%) | 134(93%)  | 142(99%)  |
| 快走    | 121     | 114(94%) | 111(92%)  | 120(99%)  |
| 慢跑    | 105     | 97(92%)  | 100(95%)  | 103(98%)  |
| 快跑    | 83      | 77(93%)  | 76(91%)   | 81(97%)   |

**结论**：

分析表1数据可得，单一波峰检测、单一过零点检测与多阈值检测的步数准确率分别可达92%、91%与97%。由此可知，通过本文提出的多阈值步数检测算法的准确率优于单一的峰值检测和过零点检测，可以有效克服单一峰值检测算法把伪波峰误判为正常行走步数的局限性，从而实现对人员步行的步数准确检测。

## 二、坐姿/站立检测算法

### 2.1 基于身体倾角的姿态检测算法

**论文来源**：

- 论文标题：《一种基于加速度传感器的人体跌倒识别方法》
- 作者：卢先领、王洪斌、王莹莹、徐保国
- 单位：江南大学物联网工程学院，江苏无锡214122
- 发表期刊：《计算机应用研究》，2013年，第30卷，第4期
- DOI：10.3969/j.issn.1001-3695.2013.04.040

**基金项目**：国家自然科学基金资助项目（30971689）；中央高校基本科研业务费专项资金资助项目（JUSRP21129）

**具体算法描述**：

本文将加速度传感器放置于人体腰腹部位置，可以检测出人体上身倾角。通过公式(8)计算可得加速度传感器的三个反映轴与水平方向夹角φX、φY和φZ，从而推测出人体所处的大致姿态。

设水平面方向为0°，当人体处于站立姿态时，+Z轴方向（人体下方）与水平面方向夹角基本稳定在－90°～－70°之间（这一角度范围也包含部分坐姿）。当人体处于坐姿时，身体往往会向一个方向微倾，+Z轴方向与水平面方向夹角大致为－70°～－30°之间。当人体处于躺卧姿态时，+Z轴方向与水平方向夹角大约为－30°～＋30°之间，同时配合X和Y轴的输出数据，还可区分前向跌倒、后向跌倒和侧向跌倒。

**人体站、坐、卧三种姿态的倾角范围**：

| 人体姿态 | +Z轴方向与水平面之间的夹角 |
|---------|---------------------------|
| 站立     | －90° ～ －70°            |
| 坐姿     | －70° ～ －30°            |
| 躺卧     | －30° ～ ＋30°            |

### 2.2 倾角计算公式

**论文来源**：同2.1节

**具体公式**：

加速度传感器输出电压与倾角的关系如公式(7)所示：

**公式7：加速度传感器输出电压与倾角关系**

```
Vo = v + (S × G × sin φ)
```

其中：

- Vo：加速度传感器的输出电压（单位：V）
- v：零加速度时传感器的输出电压（单位：V）
- S：加速度传感器的灵敏度（单位：V/g或mV/g）
- G：标准重力加速度（9.8 m/s²或1g）
- φ：倾角（单位：度）

**公式8：倾角计算**

```
φ = arcsin((Vo - v) / (S × G))
```

或者等价形式：

```
φ = arcsin((V - v) / (S × G))
```

其中：

- V：加速度传感器的实际输出电压（单位：V）
- 其他参数同公式7

**倾角计算详细步骤**：

```
输入：三轴加速度传感器输出值（Ax, Ay, Az）
输出：三个轴与水平方向的夹角（φX, φY, φZ）

1. 将加速度值转换为电压值（如果传感器直接输出加速度值，此步骤可省略）
   Vx = Ax × 灵敏度
   Vy = Ay × 灵敏度
   Vz = Az × 灵敏度

2. 计算各轴倾角
   φX = arcsin((Vx - vx) / (S × G))
   φY = arcsin((Vy - vy) / (S × G))
   φZ = arcsin((Vz - vz) / (S × G))

3. 使用Z轴倾角判断姿态
   如果 φZ 在 [-90°, -70°] 范围内：判定为站立
   如果 φZ 在 [-70°, -30°] 范围内：判定为坐姿
   如果 φZ 在 [-30°, +30°] 范围内：判定为躺卧
```

### 2.3 二次识别机制

**论文来源**：同2.1节

**具体算法描述**：

为避免跌倒后由于人体运动而导致角度变化较大，采用"疑似"跌倒后1s数据（即50个数据点）计算反映轴角度，将其平均值作为人体倾角进行二次识别。

设定阈值1为＋30°，阈值2为－30°，阈值3为－90°。加入角度信息后：快速坐下后人体上身保持竖直或向某一方向微倾，放置于人体腰腹位置的加速度传感器+Z轴与水平面夹角都在－90°～－30°之间。跌倒后上身与水平面夹角应在－30°～＋30°之间。

根据表1，坐和站立时人体倾角在－90°～－30°之间。跌倒后人体成躺卧状态，上身应保持大致水平，+Z轴与水平面夹角范围在－30°～＋30°之间。

**二次识别算法**：

```
输入：疑似跌倒数据（50个数据点）
输出：最终姿态判定

1. 计算50个数据点的Z轴倾角
   for i = 1 to 50:
       φZ[i] = 计算当前点的Z轴倾角

2. 计算平均倾角
   φ平均 = (ΣφZ[i]) / 50

3. 二次判断
   如果 φ平均 > 30° 或 φ平均 < -30°：
       判定为跌倒（躺卧状态）
   否则：
       判定为非跌倒（可能是快速坐下）
```

## 三、跌倒检测算法

### 3.1 基于HMM和身体倾角的跌倒检测算法

**论文来源**：

- 论文标题：《一种基于加速度传感器的人体跌倒识别方法》
- 作者：卢先领（1972—），男，浙江临海人，副教授，硕导，博士；王洪斌（1987—），男，江苏徐州人，硕士研究生；王莹莹（1988—），女，山东泰安人，硕士研究生；徐保国（1951—），男，江苏淮阴人，教授，博导
- 单位：江南大学物联网工程学院
- 发表期刊：《计算机应用研究》，2013年，第30卷，第4期

**具体算法描述**：

该算法首先将加速度传感器放置于人体腰腹位置，采集人在运动时的加速度变化数据；然后使用日常活动数据训练隐马尔科夫模型（HMM），利用老年人活动状态相对较少的特点，从测量数据与HMM的匹配程度寻找"疑似"跌倒行为；最后计算短暂时间内的身体倾角，检测人体躺卧姿态，完成跌倒识别。

**算法流程**：

**步骤1：数据采集与处理单元划分**

- 采样频率：50Hz
- 数据处理单元时间：3s
- 每个数据处理单元数据点：150个（50Hz × 3s = 150）
- 数据处理单元重叠率：50%

**公式1：合加速度计算**
om = √(Xm² + Ym² + Zm²)

其中：

- om：第m个采样点的合加速度值（单位：m/s²或g）
- Xm、Ym、Zm：三轴加速度传感器在X轴、Y轴、Z轴上的第m个采样数据（单位：m/s²或g）
- m：采样点序号（0 ≤ m ≤ N，N为采样点总数）

**公式2：数据处理单元划分**
Ok = {on, on+1, on+2, …, on+149}

其中：

- Ok：第k个数据处理单元的数据集
- n = 75 × (k - 1)
- k：数据处理单元序号（1 ≤ k ≤ M，M为数据处理单元总数）

**步骤2：HMM训练**

采集五种较常见的日常活动（ADL）：躺、坐、站立、行走、快速坐下训练HMM，每种行为重复50次，每次采用2个数据处理单元，则共有500个数据处理单元参与模型训练。

**HMM前向算法初始化**：

**公式3：前向变量初始化**
α1(i) = πi × bi(o1)，1 ≤ i ≤ N

其中：

- αt(i)：前向变量，表示在时刻t、状态为i时生成观测序列的概率
- πi：状态为i的初始状态概率矢量
- bi(o1)：在状态i下生成观测值o1的概率
- N：HMM状态数

**步骤3：跌倒行为识别**

给定经过训练的HMM，通过前向算法计算某一观察序列的生成概率。如果产生这一观察序列的行为来自于上述五种ADL中，则这一概率值较大，HMM与这种行为的匹配程度较高。如果这一行为是跌倒，则概率较小，匹配程度较低。利用这一概率，通过设定合适的阈值可以判断某一未知行为是否为ADL，若不是，则为"疑似"跌倒行为，然后加入角度信息再次判断。

**公式4：HMM生成概率计算**
P(O|λ) = ΣαT(i)

其中：

- P(O|λ)：给定模型λ生成观测序列O的概率
- λ：HMM模型参数
- αT(i)：时刻T的前向变量

**步骤4：基于倾角的二次识别**

为避免跌倒后由于人体运动而导致角度变化较大，采用"疑似"跌倒后1s数据（即50个数据点）计算反映轴角度，将其平均值作为人体倾角进行二次识别。

**公式5：平均倾角计算**
φ平均 = (φ1 + φ2 + … + φ50) / 50

其中：

- φ平均：反映轴与水平方向夹角的平均值（单位：度）
- φ：反映轴与水平方向夹角（单位：度）

**跌倒判定准则**：

- 如果φ平均 > 30° 或 φ平均 < -30°：判定为跌倒
- 否则：判定为非跌倒（可能是快速坐下）

### 3.2 基于三轴加速度的跌倒检测方法

**论文来源**：

- 论文标题：《一种基于三轴加速度的跌倒检测方法》
- 作者：李争
- 单位：西安电子科技大学光电工程学院，陕西西安710071
- 发表期刊：《电子科技》，2015年，第28卷，第3期
- DOI：10.16180/j.cnki.issn1007-7820.2015.03.003

**具体算法描述**：

该方法使用MMA7260Q加速度传感器，当跌倒发生时，三轴信号涉及一个巨大的信号区域，因此该方法使用加速度峰值和加速度能量曲线来产生三轴加速度信号。该方法可以避免跌倒发生时角度变化引起的误差。根据实验结果，使用本文提出的方法，跌倒检测的准确率可达90%以上。

**跌倒检测特征参数**：

**公式1：合加速度计算**
RA = √(ax² + ay² + az²)

其中：

- RA：合加速度（单位：g或m/s²）
- ax：X轴加速度（单位：g或m/s²）
- ay：Y轴加速度（单位：g或m/s²）
- az：Z轴加速度（单位：g或m/s²）

**公式2：加速度变化率**
RS = (1/t) × [∫₀ᵗAx(t)dt + ∫₀ᵗAy(t)dt + ∫₀ᵗAz(t)dt]

其中：

- RS：加速度变化率（单位：g/s或m/s³）
- Ax(t)、Ay(t)、Az(t)：三轴加速度随时间变化的函数
- t：时间（3s）

**公式3：加速度能量**
E = ∫s(t)²dt = ∫S(ω)²dω = ∫∫P(t, ω)dωdt

其中：

- E：加速度信号能量
- s(t)：时域加速度信号
- S(ω)：频域加速度信号
- P(t, ω)：时频分布

**公式4：加速度差值**
D = √[Σ(xi - yi)²]，i = 1,2

其中：

- D：加速度差值（单位：g或m/s²）
- xi、yi：不同轴的加速度值

**公式5：动态阈值**
Dth = dmin + (dmax - dmin) × 120%

其中：

- Dth：动态阈值（单位：g或m/s²）
- dmin：最小差值（单位：g或m/s²）
- dmax：最大差值（单位：g或m/s²）

**跌倒判定阈值**（基于实验数据）：

- RA > 1.85g：合加速度阈值
- SA > 2.2：加速度变化率阈值
- E > 3.5：加速度能量阈值
- Dip > 0.3：方向变化阈值

**跌倒检测算法**：

```
输入：三轴加速度数据（Ax, Ay, Az）
输出：跌倒判定结果（是/否）

1. 计算合加速度
   RA = sqrt(Ax^2 + Ay^2 + Az^2)

2. 计算加速度变化率
   RS = (1/3) * (integral(Ax) + integral(Ay) + integral(Az))

3. 计算加速度能量
   E = sum(Ax^2 + Ay^2 + Az^2)

4. 计算方向变化
   D = sqrt(sum((Ax[i] - Ay[i])^2))

5. 跌倒判定
   if (RA > 1.85 && SA > 2.2 && E > 3.5 && Dip > 0.3):
       return 跌倒
   else:
       return 非跌倒
```

### 3.3 基于AdaBoost的跌倒检测算法

**论文来源**：

- 论文标题：《基于加速度传感器的人体姿态识别研究》
- 作者：李毅
- 单位：西安邮电大学，陕西西安710100
- 发表期刊：《民间科技》，2022年（根据论文内容推断）
- 中图分类号：TP212

**具体算法描述**：

该算法使用AdaBoost算法进行人体跌倒检测。AdaBoost算法通常由多个弱分类器组成，实际预测时使用多个弱分类器进行分别预测投票后获得结果。同时，算法还会通过加权的方式输出一个强分类器，并以此来对算法进行分类判断。

**AdaBoost强分类器公式**：

H(x) = sign(Σat × ft(x))

其中：

- H(x)：强分类器输出
- at：弱分类器权重
- ft(x)：弱分类器
- T：弱分类器数量
- sign()：符号函数

**AdaBoost错误率公式**：

εt = Σwi × I(yi ≠ ft(xi))

其中：

- εt：第t次迭代的错误率
- wi：样本权重
- yi：类别标签
- I()：指示函数

**弱分类器权重公式**：

at = (1/2) × ln((1 - εt) / εt)

**权重更新公式**：
wi+1(i) = wi(t) × exp(-at × yi × ft(xi)) / Zt

其中：

- Zt：归一化因子，Zt = Σwi(t) × exp(-at × yi × ft(xi))

**跌倒识别实验结果**：

根据实验数据，跌倒识别模型的识别正确率为98%，模型训练集最好得分为89.78%。
