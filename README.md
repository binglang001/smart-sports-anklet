## 摘要

本项目设计并实现了一套基于行空板M10的智能运动腰带系统，系统采用穿戴式设计方案，将传感器模块集成于腰带之上，实现对人体运动状态、姿态以及跌倒行为的实时监测。系统以行空板M10为核心控制器，通过板载六轴传感器（ICM20689）采集三轴加速度和三轴陀螺仪数据，结合基于学术论文实现的步数检测、姿态识别和跌倒检测算法，实现高精度的运动监测功能。项目采用前后端分离架构，前端提供Web控制界面实现远程监控与数据可视化，后端采用Flask框架构建RESTful API服务。

**关键词**：行空板M10；穿戴式设备；步数检测；姿态识别；跌倒检测；多特征融合

## 一、功能特性

### 1.1 多模式运动监测

系统支持四种工作模式，满足不同场景下的使用需求：

| 模式 | 编号 | 功能描述 |
|------|------|----------|
| 生活模式 | 0 | 姿态监测、久坐提醒、跌倒检测、温湿度监测 |
| 运动模式 | 1 | 步数统计，配速计算、碳排放追踪、运动时长记录 |
| 会议模式 | 2 | 屏幕休眠、静音、保持基础数据采集 |

### 1.2 健康数据监测

系统集成了多类型传感器，实现全方位的健康数据采集：

| 传感器类型 | 型号/规格 | 监测内容 | 采样频率 |
|-----------|----------|---------|---------|
| 温湿度传感器 | DHT11 | 环境温度、湿度 | 5秒/次 |
| 六轴传感器 | ICM20689 | 三轴加速度、三轴陀螺仪 | 50Hz |
| 旋钮电位器 | 模拟量 | LED亮度调节 | 100ms |
| 按钮 | 数字量 | 模式切换、紧急呼叫 | 实时 |

### 1.3 智能提醒功能

- **久坐提醒**：用户长时间保持坐姿时，系统自动语音提示
- **跌倒警报**：检测到跌倒行为时，触发紧急救援流程
- **消息推送**：Web端可向设备发送自定义消息，语音播报

## 二、核心算法

本项目的核心算法完全基于学术论文实现，所有算法内容均直接来源于论文原文。

### 2.1 步数检测算法

#### 2.1.1 算法来源

参考论文：
- **《基于腰部MEMS加速度计的多阈值步数检测算法》** 作者：蒋博、付乐乐（重庆邮电大学光电工程学院）
- **《分段双向去除反向重力加速度算法》** 作者：李兴、侯振杰、梁久祯、常兴治（常州大学信息科学与工程学院）

#### 2.1.2 重力加速度去除

由于近地面地球引力的影响，加速度传感器采集到的传感器加速度是反向重力加速度与线性加速度的矢量和。只有线性加速度与人体的运动相关，反向重力加速度的存在导致传感器加速度与人体运动的关联性不高。

本系统采用**双向角度法**去除重力加速度：

```
1. 静止点检测：检测加速度接近0的数据点作为静止点
2. 基于静止点数据分段：以静止点为端点将运动数据分段
3. 双向角度法：
   - 从前往后积分计算反向重力加速度得到线性加速度as
   - 从后往前积分计算反向重力加速度得到线性加速度at
   - 取平均：a(i) = (as + at) / 2
```

静止点判断条件：|a'| < 0.1g

#### 2.1.3 多阈值步数检测算法

采用三阶段检测方法：

**步骤1（初始化）**：
- 窗口大小：7
- 初始步数：Step = 0
- 初始标志位：Flag = 1

**步骤2（波峰检测）**：
- 条件1：C[0] > T_max 且 C[1] > T_max 且 ... 且 C[6] > T_max
- 条件2：C[3] > C[0] 且 C[3] > C[1] 且 ... 且 C[3] > C[6]
- 同时满足：Flag = 2

**步骤3（过零点检测）**：
- 条件1：C[3] < 0
- 条件2：C[2] > 0
- 同时满足：Flag = 3

**步骤4（波谷检测）**：
- 条件1：C[0] < T_min 且 C[1] < T_min 且 ... 且 C[6] < T_min
- 条件2：C[3] < C[0] 且 C[3] < C[1] 且 ... 且 C[3] < C[6]
- 同时满足：Flag = 1，Step = Step + 1

**步骤5（步频验证）**：
- 最小步间隔：250ms
- 最大步间隔：2000ms

#### 2.1.4 核心参数

| 参数名称 | 默认值 | 说明 |
|---------|--------|------|
| t_max | 1.5g | 波峰阈值 |
| t_min | -0.5g | 波谷阈值 |
| window_size | 7 | 滑动窗口大小 |
| min_interval_ms | 250 | 最小步间隔 |
| max_interval_ms | 2000 | 最大步间隔 |

#### 2.1.5 实验结果

根据论文《基于腰部MEMS加速度计的多阈值步数检测算法》的实验数据：

| 运动模式 | 实际步数 | 多阈值检测 |
|---------|---------|-----------|
| 慢走    | 157     | 156(99%)  |
| 正常行走 | 143     | 142(99%)  |
| 快走    | 121     | 120(99%)  |
| 慢跑    | 105     | 103(98%)  |
| 快跑    | 83      | 81(97%)   |

### 2.2 姿态识别算法

#### 2.2.1 算法来源

参考论文：**《一种基于加速度传感器的人体跌倒识别方法》** 作者：卢先领、王洪斌、王莹莹、徐保国（江南大学物联网工程学院）

#### 2.2.2 姿态角计算

利用三轴加速度计算俯仰角（pitch）和翻滚角（roll）：

```
俯仰角: pitch = atan2(ay, √(ax² + az²)) × 180/π
翻滚角: roll  = atan2(ax, √(ay² + az²)) × 180/π
```

#### 2.2.3 姿态判定规则

根据论文中Z轴与水平面夹角判断姿态：

| 姿态 | 翻滚角范围 |
|------|-----------|
| 站立 | -90° ~ -70° |
| 坐姿 | -70° ~ -30° |
| 躺卧 | -30° ~ +30° |

---

### 2.3 跌倒检测算法

#### 2.3.1 算法来源

参考论文：**《一种基于三轴加速度的跌倒检测方法》** 作者：李争（西安电子科技大学光电工程学院）

#### 2.3.2 特征提取

根据论文，提取以下四个特征：

| 特征 | 计算公式 | 阈值 |
|-----|---------|------|
| 合加速度 | RA = √(ax² + ay² + az²) | > 1.85g |
| 加速度变化率 | SA = (1/t) × ∫A(t)dt | > 2.2 |
| 加速度能量 | E = ∫s(t)²dt | > 3.5 |
| 方向变化 | D = √[Σ(xi - yi)²] | > 0.3 |

#### 2.3.3 跌倒判定

论文原文判定条件：
```
if (RA > 1.85 && SA > 2.2 && E > 3.5 && Dip > 0.3):
    return 跌倒
else:
    return 非跌倒
```

根据实验数据，该方法的跌倒检测准确率可达90%以上。
